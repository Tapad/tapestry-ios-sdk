Tapestry Mobile SDKs

Add Tapestry to your iOS and Android apps

What's Tapestry?

Tapestry is an API which allows you to store and retrieve data in Tapad's device graph, associated to the device which is making the requests. Think about it like a cookie, except information can also be retrieved from associated devices.

How do I use it?

  • Ask us for a partner ID at tapestry@tapad.com.
  • Read the iOS setup instructions and AppleDocs API.
  • Read the Android setup instructions and JavaDocs API.
