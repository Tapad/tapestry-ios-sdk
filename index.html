Summary

Tapestry iOS SDK allows you to add Tapestry functionality in your iOS apps.

What's Tapestry?

Tapestry is an API which allows you to store and retrieve data in Tapad's device graph, associated to the device which is making the requests. Think about it like a cookie, except information can also be retrieved from associated devices.

How do I use it?

  • Get a partner ID (tapestry@tapad.com)
  • git clone https://github.com/Tapad/tapestry-ios-sdk.git
  • Drag and drop the Tapestry iOS project into your iOS project
  • Follow the setup instructions in the readme here https://github.com/Tapad/tapestry-ios-sdk
